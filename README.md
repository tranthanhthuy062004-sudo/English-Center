# English Center E-learning System

> Business Analyst-oriented academic web project for managing an English learning center with role-based workflows, online learning features, dashboards, assessments, and AI-supported learning.

## Project Overview

English Center is a full-stack academic web project designed to support the operation of an English learning center.

The system focuses on 3 main user roles:

- Student
- Teacher
- Administrator

The project covers core workflows such as authentication, course management, online assessment, dashboard tracking, learning material management, notification, and AI-supported learning features.

## Key Features

### Student

- Register and log in to the system.
- View available courses and learning content.
- Complete online exercises and assessments.
- Track learning progress through dashboard.
- Use AI-supported learning features.

### Teacher

- Manage learning materials and exercises.
- View assigned classes and students.
- Support assessment and learning activities.
- Submit or manage teaching content.

### Administrator

- Manage students, teachers, courses, and learning content.
- Review system data through dashboard.
- Manage reports, notifications, and system operations.
- Monitor learning and operational workflows.
